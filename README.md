# Desafio UI

Frontend do desafio Duxus, desenvolvido em **Angular 19** com **PrimeNG**, **TailwindCSS** e integração com o backend via **proxy**.

---

## ⚙️ Requisitos

- Node.js 18+
- npm 9+
- Angular CLI 19+

---

## 📦 Instalação

1. Instale as dependências do projeto:

```bash
npm install
```

## ▶️ Rodando em modo desenvolvimento
```bash
npm start
```
